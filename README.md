## 簡介
```sh
此App是一款以寵物為主的社群軟體，使用者可以透過程式向好友們發布寵物相關的貼文，讓其他使用者留言、分享，並且會在每天固定時間進行好友抽卡，擴增
使用者的好友圈，除此之外使用者還可以瀏覽與寵物相關的活動，並透過此App進行報名便能與寵物一同參加活動，雙向寵物翻譯功能，讓使用者與寵物平時的
相處中能更添樂趣，讓飼主能了解寵物想表達的意思並加深與寵物間的羈絆。
```
